# Python作业
## 一、核心执行文件路径及功能
1. **数据预处理文件**：`src/01_data_preprocessing.py`
   对原始数据进行缺失值填充、异常值检测与处理，完成正态性检验、共线性诊断（VIF计算）及描述性统计，输出清洗后的标准化数据，为后续分析奠定基础。
2. **PCA与因子分析文件**：`src/02_pca_factor_analysis.py`
   基于预处理数据执行PCA降维（自动选择最优主成分数），完成因子分析适用性检验（KMO/Bartlett），提取公因子并输出载荷矩阵、得分矩阵，实现数据维度简化与特征提取。
3. **聚类与分类分析文件**：`src/03_clustering_classification.py`
   以PCA得分数据为基础，通过肘部法则+轮廓系数确定最优K值，执行KMeans聚类；结合聚类标签构建逻辑回归分类模型，评估聚类结果的可解释性与区分度。

## 二、项目说明
本项目聚焦多元统计分析方法实践，以数字经济相关数据为样本，覆盖数据预处理、降维分析、聚类分类全流程。核心目标是验证多元统计方法在数据特征挖掘中的应用效果，适用于课程大作业、多元统计分析入门实践场景。
