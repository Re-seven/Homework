# Python多元统计作业
## 一、项目目录结构及文件说明
### 1. 核心代码目录：`src/`
存放所有可执行的Python分析脚本，各文件功能如下：
- `01_data_preprocessing.py`（数据预处理）：对原始数据做缺失值填充、异常值检测处理，完成正态性检验、共线性诊断（VIF计算）及描述性统计，输出清洗后的标准化数据，为后续分析奠定基础。
- `02_pca_factor_analysis.py`（PCA与因子分析）：基于预处理数据执行PCA降维（自动选择最优主成分数），完成因子分析适用性检验（KMO/Bartlett），提取公因子并输出载荷矩阵、得分矩阵，实现数据维度简化与特征提取。
- `03_clustering_classification.py`（聚类与分类分析）：以PCA得分数据为基础，通过肘部法则+轮廓系数确定最优K值，执行KMeans聚类；结合聚类标签构建逻辑回归分类模型，评估聚类结果的可解释性与区分度。

### 2. 数据目录：`data/`
- `data/raw/`：存放项目原始数据集（如数字经济相关CSV文件），为分析提供基础数据源。
- `data/processed/`：存放经`01_data_preprocessing.py`处理后的清洗数据、标准化数据、共线性处理后最终数据等，供后续PCA、聚类分析调用。

### 3. 结果输出目录：`results/`
存放各分析环节生成的可视化图表（如缺失值热力图、PCA方差贡献率图、聚类结果分布图）、数据结果文件（如VIF检验结果.csv、因子得分矩阵.csv）及模型评估报告，所有输出文件按分析环节分类存放。

### 4. 依赖清单文件：`requirements.txt`
存放运行所有代码所需的Python依赖库（如pandas、scikit-learn、factor_analyzer等），包含库名及最低兼容版本。

## 二、项目说明
本项目聚焦多元统计分析方法实践，以数字经济相关数据为样本，覆盖数据预处理、降维分析、聚类分类全流程。核心目标是验证多元统计方法在数据特征挖掘中的应用效果，适用于课程大作业、多元统计分析入门实践场景。
